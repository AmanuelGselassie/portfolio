Margot Menu

Margot is a dynamic menu for restaurants. it is a mobile-first but screen-responsive website.

The concept of Margot is to make the user experience easy and smooth. The flow on Margot is a user will go to the desired restaurant and scan a specific QR code which is provided by Margot which will redirect him to the restaurant's dynamic menu. choose an item, see the reviews given by other users and order.
